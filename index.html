Introducing Gladius

Headline 2

Play Our Demo see an example of what you can build

Gladius is creating building blocks for great games on the open web. It currently includes an engine with core infrastructure support and a math library. Related standalone libraries and templates will be underway soon.

Key Gladius features

  • 3D support: CubicVR; math library optimized for WebGL
  • baked-in best practices: entity-component framework, automated testing
  • built for quality using battle-tested tech: require.js, box2d.js, qunit, jshint, sinon
